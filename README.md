## Block-ManageCertificates-Chain
Este projeto apresenta uma alternativa para gerenciamento - emissão, visualização e controle de acesso - de certificado de conclusão de curso de forma descentralizada e distribuida. O principal objetivo é ferramentaliza o cliente final (aluno) com acesso total e irrestrito aos seus dados escolares.

## Tecnologias Utilizadas
* Para controle de emissão e prova de autenticidade dos documentos emitidos: **Blockchain Ethereum**
* Para armazenamento fisico dos documentos e prova de autencidade do conteúdo dos documentos emitidos: **IPFS**
